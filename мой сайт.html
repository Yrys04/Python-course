Что такое Python?

Python — это высокоуровневый язык программирования, известный своей простотой и читаемостью.

Основной синтаксис

Python использует отступы для обозначения блоков кода, что делает код чистым и структурированным.

print("Привет, мир!")

Примеры кода

Рассмотрим простой цикл в Python:

for i in range(5):
    print(i)
